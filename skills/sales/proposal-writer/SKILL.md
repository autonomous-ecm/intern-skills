# Proposal Writer

## Description
Draft sales proposals/quotations: customize based on client needs, highlight value proposition, and close the deal.

## Trigger
- User needs to write a sales proposal / quotation
- User says "proposal", "quotation", "quote", "pitch", "pitch deck"
- User is preparing documents for a deal

## Instructions

### When to Activate
- Write a proposal for a potential customer
- Create a custom quotation based on requirements
- Prepare a pitch deck / presentation
- Follow up on a previously sent proposal
- Customize a template by industry/client

### Proposal Structure
1. **Cover**: Logo, client name, date, project name
2. **Executive Summary**: 1 page, summarizing the problem and solution
3. **Understanding**: Demonstrate understanding of client needs (pain → impact)
4. **Solution**: Detailed solution, features, benefits
5. **Why Us**: USP, case studies, testimonials
6. **Pricing**: Clear pricing table, options/packages
7. **Timeline**: Implementation roadmap
8. **Next Steps**: Specific CTA, contact info
9. **Appendix**: Detailed specs, terms & conditions

### Process
1. Gather info: client needs, budget range, decision criteria, competitors
2. Customize: tailor each section to the client, do not use generic content
3. Pricing: propose 3 options (Good / Better / Best)
4. Review: proofread, check numbers, ensure consistency
5. Deliver: professional PDF, accompanied by a cover email

### Rules
- Lead with value, not features — "You will save 30% on costs" > "The product has feature X"
- Personalize: mention the client's name, industry, specific pain points
- 3 pricing tiers: guide the client toward the middle option (anchoring)
- Deadline: proposal is valid for 30 days (create urgency)
- Follow up: day 3, day 7, day 14 after sending

### Output Format

🏢 Client: [Company name]
