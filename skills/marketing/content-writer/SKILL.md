# Content Writer

## Description
Write marketing content: blog posts, social media posts, ad copy, landing page copy, and newsletters.

## Trigger
- User needs to write marketing content
- User says "write article", "content", "blog", "copywriting", "caption", "ad copy"
- User needs content ideas

## Instructions

### When to Activate
- Write a blog post / article
- Compose social media captions (Facebook, Instagram, LinkedIn, TikTok)
- Write ad copy (Google Ads, Facebook Ads)
- Landing page copywriting
- Newsletter / email marketing content
- Product description

### Content Types
- **Blog post**: 800-1500 words, SEO-optimized, informative/educational
- **Social caption**: 50-300 words, engaging, clear CTA
- **Ad copy**: Headline + Description, attention-grabbing, conversion-focused
- **Landing page**: Hero → Benefits → Social proof → CTA
- **Newsletter**: Subject + Preview + Body, value-driven
- **Product desc**: Features → Benefits → Use cases

### Process
1. Brief: Purpose, target audience, tone, key message, CTA
2. Research: Competitor content, trending topics, keyword research
3. Outline: Structure the article
4. Write: First draft
5. Optimize: SEO, readability, CTA placement
6. Review: Proofread, fact-check

### Rules
- Always ask about target audience and tone before writing
- Strong hook in the first sentence — 80% of people only read the headline
- AIDA framework: Attention → Interest → Desire → Action
- Write for scanners, not readers: headings, bullets, short paragraphs
- Each piece of content must have exactly 1 clear CTA
- Avoid jargon unless the target audience is experts

### Output Format
```
✍️ Content Draft

📌 Type: [Content type]
🎯 Audience: [Target]
🎨 Tone: [Tone of voice]

---
[Content body]
---

📊 Stats:
- Words: [Word count]
- Readability: [Grade level]
- SEO keywords: [Keywords used]

💡 Variations: [2-3 alternative headlines/hooks if needed]
```
