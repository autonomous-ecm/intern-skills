# SEO Optimizer

## Description
Optimize SEO: keyword research, on-page SEO audit, meta tags, content optimization, and technical SEO check.

## Trigger
- User needs to optimize SEO for a website/content
- User says "SEO", "keyword", "ranking", "meta tag", "search engine", "organic traffic"
- User wants to audit current SEO

## Instructions

### When to Activate
- Keyword research for new content
- On-page SEO audit for a web page
- Optimize meta title, description, headings
- Content SEO: keyword density, internal linking
- Technical SEO checklist

### On-Page SEO Checklist
- [ ] Title tag: 50-60 chars, keyword at the beginning
- [ ] Meta description: 150-160 chars, includes CTA
- [ ] H1: Only 1, contains primary keyword
- [ ] H2-H6: Used appropriately, contain related keywords
- [ ] URL: Short, readable, contains keyword
- [ ] Image alt text: Descriptive + keyword
- [ ] Internal links: 3-5 links to related pages
- [ ] External links: 1-2 links to authoritative sources

### Process
1. Define the goal: which page/keyword needs optimization
2. Keyword research: main keyword + related + long-tail
3. Analyze SERP: top 10 currently ranking → understand search intent
4. On-page audit: check each factor
5. Content optimization: natural usage, no keyword stuffing
6. Technical check: speed, mobile, schema markup

### Rules
- Search intent first: understand what the user wants when searching that keyword
- Keyword density of 1-2% is sufficient — NO keyword stuffing
- Content must be valuable to readers, not just bots
- Mobile-first: check responsive design, Core Web Vitals
- Update old content: fresh content ranks better
