# FAQ Lookup

## Description
Quickly look up frequently asked questions, find standardized answers, and update the FAQ when new information is available.

## Trigger
- User asks a common customer question
- User says "FAQ", "frequently asked questions", "customers often ask", "canned response"
- User needs a quick template answer

## Instructions

### When to activate
- Find an answer for a common question
- Create or update a new FAQ entry
- Customer asks a question already in the FAQ → respond quickly
- Compile new recurring questions → add them to the FAQ

### FAQ categories
- **Products**: Features, specifications, comparisons, warranty
- **Orders**: Tracking, delivery time, order modifications
- **Payments**: Payment methods, refunds, promotions
- **Account**: Registration, login, password reset
- **Technical**: Setup, usage, troubleshooting
- **Policies**: Returns, warranty, refunds, privacy

### Process
1. Receive a question → search the FAQ database
2. If found → provide the standardized answer, customize to the context
3. If not found → draft a new answer, suggest adding it to the FAQ
4. Track question frequency → prioritize FAQ for the most asked questions

### Rules
- FAQ answers must be concise and easy to understand (no jargon)
- Update the FAQ when policies/products change
- Each FAQ entry should have: question, short answer, detailed answer, reference link
- Tag FAQ entries by category for easy searching
- Review FAQs monthly: remove outdated entries, add new ones

### Output Format
```
❓ FAQ Match

**Q: [Question]**

**A (Short):**
[1-2 sentence answer]

**A (Detailed):**
[Full explanation with step-by-step if needed]

🔗 Reference: [Link]
🏷️ Category: [Category]
📅 Last updated: [Date]
```
