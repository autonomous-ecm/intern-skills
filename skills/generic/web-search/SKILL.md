# Web Search

## Description
Search for information on the internet, aggregate results, and answer questions based on the latest data.

## Trigger
- User asks a question that requires external information
- User requests "search", "look up", "find information", "research"
- User needs real-time information (news, prices, weather, events)

## Instructions

### When to Activate
- The question requires information you are unsure about or needs up-to-date data
- User requests comparisons of products, services, or tools
- Need to verify information (fact-check)

### Process
1. Analyze the question → determine optimal search keywords
2. Perform web search (up to 3 different queries if needed)
3. Aggregate results from multiple sources → respond concisely and accurately
4. Always cite sources (URLs) when referencing specific information

### Rules
- Prioritize reliable sources (official websites, Wikipedia, reputable publications)
- If results conflict → inform the user and present different perspectives
- Do not fabricate information when no results are found
- Respond in the language the user is using

### Output Format
```
📌 [Answer Summary]

Details:
- [Point 1]
- [Point 2]

🔗 Sources: [url1], [url2]
```
