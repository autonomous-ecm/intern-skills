# Performance Review

## Description
Support the performance evaluation process: create review forms, consolidate feedback, write reviews, and track OKRs/KPIs.

## Trigger
- It is performance review season
- User says "evaluation", "performance review", "KPI", "OKR", "employee review"
- User needs to write or consolidate feedback

## Instructions

### When to activate
- Create a periodic performance review form
- Write a self-review or review for a team member
- Consolidate 360-degree feedback
- Set up and track OKRs/KPIs
- Prepare content for a 1:1 review meeting

### Review Process
1. **Goal Setting** (beginning of cycle): Define clear, measurable OKRs/KPIs
2. **Mid-cycle Check-in**: Review progress, adjust if needed
3. **Self-review** (end of cycle): Employee self-assessment
4. **Manager Review**: Manager evaluation + feedback
5. **Calibration**: Score calibration across teams
6. **1:1 Discussion**: Discuss results, plan development

### Evaluation Criteria
- **Performance (60%)**: KPI/OKR completion, quality of work
- **Competency (20%)**: Technical skills, problem solving, communication
- **Culture (20%)**: Teamwork, initiative, company values

### Rating Scale
- **5 - Outstanding**: Far exceeds expectations
- **4 - Exceeds**: Exceeds expectations
- **3 - Meets**: Meets expectations
- **2 - Below**: Below expectations
- **1 - Needs Improvement**: Significant improvement needed

### Rules
- Feedback must be specific with examples (no vague statements)
- Use the SBI framework: Situation -> Behavior -> Impact
- Balance strengths and areas for improvement
- Development plans must include specific actions and timelines
- Keep evaluation information confidential

### Output Format
```
📊 Performance Review - [Employee Name]
📅 Review Period: [Q/Year]
🎯 Position: [Title]

## Overall Score: [X/5] - [Rating Text]

| Criteria | Score | Comments |
|-----------|-------|----------|
| Performance | [X/5] | [SBI feedback] |
| Competency | [X/5] | [SBI feedback] |
| Culture | [X/5] | [SBI feedback] |

## Strengths
- [Strength + specific example]

## Areas for Improvement
- [Area + specific example]

## Development Plan
- [ ] [Action] - Deadline: [date]
```
